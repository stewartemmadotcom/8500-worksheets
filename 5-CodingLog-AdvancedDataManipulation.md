### Challenge 2: Lubridate confused me 

**What I was trying to do:**
I was trying to use lubridate to create R readable dates on a vector of differently formatted dates. 

**What went wrong:**
Only some of the dates were being parsed, not all of them. 

**My problem-solving process:**
1. I tried to use different variations, including mdy, ymd, dmy. 
2. I looked at the documentation for lubridate and then did some googling. 
3. [Additional attempts if relevant]

**Resources I consulted:**
- [X] Documentation for [package/function]
- [ ] Stack Overflow: [describe what you searched for]
- [ ] Course materials: [which ones]
- [ ] Slack discussion
- [ ] Office hours
- [X] Other: I googled why only some of my dates were being parsed. 

**Resolution:**
I realized that lubridate only changes dates into the R readable version, which is YMD. I also realized that I needed to have the same format for all the dates I was trying to convert. 

**What I learned:**
I was previously thinking about lubridate the wrong way; I thought that I was using the function to change dates to the mdy, ydm, or ymd formats. Instead, I was telling R what format the date was ALREADY in and having it changed to the YMD format.
